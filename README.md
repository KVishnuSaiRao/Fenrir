This production-grade React application, built with Next.js and Tailwind CSS v4, provides an expert-level cybersecurity interface for rapid target mapping and vulnerability reporting.


## Technical Stack
Framework: Next.js (App Router)

Language: TypeScript

Styling: Tailwind CSS v4.0

Icons: Lucide React

## Setup & Deployment
Install dependencies:
npm install

Run locally:
npm run dev

Project Link: https://aps-secure.vercel.app/
